Learn Frontend Styling

Interactive playgrounds to learn frontend concepts by playing with them.

Pick a topic from the sidebar to get started. Currently available:

  • Learn CSS Flex — a live flexbox playground: tweak every flex property and watch the layout (and the CSS code) update in real time.
  • Learn CSS Grid — a live grid playground: build track layouts, gaps, alignment, and spanning items, with generated CSS as you go.
  • Learn CSS Positioning — explore static, relative, absolute, fixed, and sticky positioning, with offsets and z-index, and the CSS updating live.
  • Learn CSS Centering — a technique picker (text-align, margin auto, flex, grid, absolute tricks…) from the CSS-Tricks complete guide, each with live preview and generated CSS.
  • Learn CSS Cascade Layers — watch multiple rules compete for one element's color, reorder @layers, toggle !important, and read a live cascade breakdown explaining exactly which rule wins — and why.
